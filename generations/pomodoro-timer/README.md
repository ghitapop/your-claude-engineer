# Pomodoro Timer

A minimal, beautiful pomodoro timer web app.

## Features
- Large countdown timer with progress ring
- Start/Pause/Reset/Skip controls
- Work sessions (25 min), short breaks (5 min), long breaks (15 min)
- Session counter with localStorage persistence
- Audio notification on completion

## Tech Stack
- Vanilla HTML/CSS/JavaScript
- No backend required
- Local storage for persistence

## Getting Started
```bash
./init.sh
```
Then open http://localhost:3000

## Design
- Dark theme with tomato red accent (#ff6347)
- Mobile-friendly, centered layout
- Smooth animations
